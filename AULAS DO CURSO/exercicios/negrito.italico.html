FORMATACAO DE TEXTOS EM HTML

NEGRITO

Para escrever em negrito usamos a tag "strong"

Veja o exemplo:

ESTE É UM TEXTO EM NEGRITO ESta frase foi escrita com a tag "strong"

ITÁLICO

Para escrever em itálico usamos a tag "em".

ESTE É UM TEXTO ESCRITO EM ITÁLICO Essa frase é eecrita usando a tag "em", ficando semântica.
