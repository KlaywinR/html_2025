Outras Formatações em HTML

1.1 Escrevendo códigos fontes

Usando a tag "code" a palavra escrita fica monoespaçada:

Este é um exemplo de espaçamento!

-- Em exemplo, usamos um comando em linguagem C:

#include <stdio.h>
int main(void) {
    printf("\nOlá, mundo\n");
    return 0;
}

1.2 Citações Simples

-- Em exemplo, usaremos uma citação:

Diria o pai de um colega meu: O computador é a máquina mais burra já criada pelo ser humano.

Neste caso, se usa a tag "q" para criar uma citação

1.3 Citações Completas

Usamos a tag "blockquote" para ter uma citação completa. Veja um exemplo:

-- Livro do Steve Jobs; melhor citação: --
"Seu tempo é limitado, então não o desperdice vivendo a vida de outra pessoa."
— Steve Jobs

-- Veja, a citação completa fica destacada no texto.
